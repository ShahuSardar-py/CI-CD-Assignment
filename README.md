# ID Card Webpage

A simple Node.js application that serves an ID card webpage.

## Installation

1. Install dependencies:
   ```
   npm install
   ```

2. Start the server:
   ```
   npm start
   ```

3. Open your browser and go to `http://localhost:3000`
